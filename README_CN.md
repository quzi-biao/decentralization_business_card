# 🎴 去中心化数字名片

一个注重隐私、去中心化的数字名片应用，基于 React Native 和 Expo 构建。通过二维码安全地交换名片，支持端到端加密。

[English Documentation](./README.md)

## ✨ 功能特性

### 🔐 隐私与安全
- **端到端加密**：所有名片数据在存储前使用 AES-256 加密
- **去中心化身份**：基于 DID（去中心化标识符）系统
- **本地优先存储**：数据默认保存在您的设备上
- **可选云备份**：将加密数据上传到 MinIO 实现跨设备同步

### 📇 名片管理
- **数字名片**：创建和管理您的数字名片
- **二维码交换**：通过扫描二维码分享名片
- **联系人组织**：为联系人添加标签和分类
- **富媒体支持**：添加头像、微信二维码和公司图片
- **智能搜索**：按姓名、公司、职位或备注搜索联系人

### 🤖 AI 功能
- **AI 助手**：与 AI 助手聊天了解您的联系人
- **名片评估**：获取 AI 驱动的名片洞察
- **语音输入**：支持语音转文字的免提交互

### 🎨 用户体验
- **现代化界面**：简洁直观的界面，流畅的动画效果
- **深色模式**：支持主题切换
- **离线支持**：无需网络连接即可使用
- **网络恢复**：连接恢复时自动刷新

## 🚀 快速开始

### 前置要求

- Node.js 18+ 和 npm/yarn
- Expo CLI
- iOS 模拟器（用于 iOS 开发）或 Android Studio（用于 Android）

### 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/yourusername/business-card.git
cd business-card
```

2. 安装依赖
```bash
npm install
# 或
yarn install
```

3. 配置环境变量
```bash
cp .env.example .env
# 编辑 .env 文件并填入您的配置
```

4. 启动开发服务器
```bash
npx expo start
```

5. 在设备上运行
- 按 `i` 启动 iOS 模拟器
- 按 `a` 启动 Android 模拟器
- 使用 Expo Go 应用扫描二维码在真机上运行

## ⚙️ 配置说明

### n8n 集成（可选）

本应用使用 n8n 提供 AI 功能。要启用 AI 功能：

1. 搭建 n8n 实例（自托管或云端）
2. 创建以下工作流：
   - AI 代理（聊天助手）
   - API 密钥管理
   - 语音转文字
3. 在 `.env` 文件中更新 n8n 凭据

详细说明请参考 [n8n 配置指南](./docs/n8n-setup.md)。

### MinIO 存储（可选）

用于云存储和跨设备同步：

1. 搭建 MinIO 服务器
2. 创建名片存储桶
3. 在 `.env` 中更新 MinIO 凭据

## 📱 使用说明

### 创建名片

1. 打开应用并导航到"我的名片"
2. 填写您的信息：
   - 姓名、职位、公司
   - 联系方式（电话、邮箱、微信）
   - 上传头像和公司图片
3. 名片会自动加密并保存

### 交换名片

1. 导航到"交换"标签
2. 向他人展示您的二维码，或
3. 点击"扫描二维码"扫描他人的名片
4. 名片通过端到端加密进行交换

### 管理联系人

1. 在"名片"标签中查看所有联系人
2. 按姓名、公司或职位搜索
3. 添加标签和备注来组织联系人
4. 查看详细的联系人信息

## 🏗️ 技术架构

### 技术栈

- **前端**：React Native + Expo
- **状态管理**：Zustand
- **存储**：AsyncStorage（本地）+ MinIO（云端）
- **加密**：AES-256 加密
- **AI 后端**：n8n 工作流
- **身份系统**：自定义 DID 实现

### 项目结构

```
business-card/
├── src/
│   ├── components/      # 可复用 UI 组件
│   ├── screens/         # 页面组件
│   ├── services/        # 业务逻辑服务
│   ├── store/           # Zustand 状态管理
│   ├── utils/           # 工具函数
│   ├── config/          # 配置文件
│   └── navigation/      # 导航设置
├── assets/              # 图片、字体等
├── .env.example         # 环境变量模板
└── app.json             # Expo 配置
```

## 🔒 安全性

- 所有名片数据使用 AES-256 加密
- 私钥永不离开您的设备
- 二维码仅包含加密数据
- 可选的云存储使用加密包
- 没有中心服务器存储您的未加密数据
